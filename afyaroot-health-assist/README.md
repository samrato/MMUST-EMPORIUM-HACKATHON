# AFYAROOT Health Assist

AFYAROOT Health Assist is a Vite + React application for rural health support. It includes:

- Home dashboard
- Symptom checker
- Emergency response panel
- Chat AI assistant
- Facility discovery
- Analytics
- Booking
- Settings

## Run locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm run build
npm test
```
